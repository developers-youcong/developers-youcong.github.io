## 一、windows杀死端口进程步骤

#### 1.查询端口进程
```
netstat -ano | findstr 端口

```

#### 2.找到端口进程占用的应用
```
tasklist | findstr 进程号

```

#### 3.杀死端口进程
```
taskkill -PID 进程号 -F

```

## 二、应用场景
想杀死某个端口进程，但在服务列表中却找不到，可以之间通过命令行找到这个进程并杀死该进程。这样一来，减少重启电脑和找到问题根源。
